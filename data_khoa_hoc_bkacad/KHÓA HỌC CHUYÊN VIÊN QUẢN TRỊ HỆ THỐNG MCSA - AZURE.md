## **1. Trình độ chứng chỉ**

Chứng chỉ Microsoft Azure Administrator, Windows Server Hybrid Administrator.

Mã môn thi: Az – 104, AZ-800 & AZ-801.

## **2. Mục tiêu khóa học**

Khóa học MCSA-Azure được thiết kế nhằm cung cấp cho học viên kiến thức và kỹ năng cần thiết để quản lý, triển khai và tối ưu hóa hạ tầng Windows Server, Azure, cũng như các mô hình Hybrid Cloud.

Khóa học giúp học viên nắm vững các công nghệ hiện đại trong quản trị hệ thống, từ on-premises đến cloud, đảm bảo vận hành hệ thống ổn định, bảo mật và hiệu quả.  
Mục tiêu cụ thể của khóa học:  
•    Hiểu Biết Về Windows Server:  
-    Nắm rõ tổng quan và các tính năng mới của Windows Server.  
-    Triển khai và cấu hình các phiên bản khác nhau của Windows Server (Nano, Core, Full).  
-    Quản lý các dịch vụ nền tảng như Active Directory Domain Services (AD DS), DHCP, DNS.  
•    Quản Lý Hệ Thống Lưu Trữ:  
-    Cấu hình và quản lý công nghệ lưu trữ trong Windows Server.  
-    Làm việc với các mô hình lưu trữ như Basic Disk, Dynamic Disk, Storage Spaces.  
-    Triển khai RAID, Storage Pool và High Availability Storage.  
•    Ảo Hóa và Hệ Thống Hybrid:  
-    Triển khai Hyper-V, quản lý máy ảo và các công nghệ liên quan như Windows Server Containers, Hyper-V Containers.  
-    Quản lý snapshot, virtual switch và live migration.  
-    Tích hợp hạ tầng on-premises với Azure bằng Azure Migrate, Azure Arc.  
•    Quản Lý Mạng và Tính Sẵn Sàng:  
-    Thực thi Network Load Balancing (NLB) và Failover Clustering.  
-    Cấu hình VPN Client-to-Site, Site-to-Site và DirectAccess.  
-    Bảo mật hệ thống với Windows Defender Credential Guard, BitLocker, RBAC.  
•    Quản Lý Dịch Vụ Mạng:  
-    Cấu hình và quản lý DHCP, DNS, IPv6.  
-    Cấu hình DNS Round Robin, Netmask Ordering, Conditional Forwarder.  
-    Triển khai và quản lý Windows Deployment Services, WSUS.  
•    Quản Lý Active Directory:  
-    Quản lý các đối tượng trong Active Directory.  
-    Triển khai Group Policy, Folder Redirection, Software Deployment.  
-    Thực thi AD DS Trusts Relationship, AD DS Sites & Replication.  
-    Cấu hình Read-Only Domain Controller (RODC), Global Catalog.  
•    Bảo mật và Quản lý Chứng Chỉ:  
-    Triển khai Active Directory Certificate Services (AD CS) và Rights Management Services (AD RMS).  
-    Quản lý Certificate Templates, cấu hình bảo vệ tài liệu & email.  
-    Backup và Restore hệ thống chứng chỉ.  
•    Quản Lý Azure & Hybrid Cloud:  
-    Nắm vững quy trình quản lý tài nguyên Azure, RBAC, Azure Policy.  
-    Triển khai và quản lý Virtual Machines (VMs), cấu hình networking và backup.  
-    Quản lý Azure File Sync, Azure Storage, Azure Migrate.  
-    Giám sát hệ thống Hybrid với Azure Monitor, Metrics Explorer.  
•    Quản Lý Azure Active Directory & Hybrid AD:  
-    Quản lý đối tượng trong Azure AD, triển khai Hybrid AD.  
-    Thực thi Seamless Single Sign-on (SSO) và MFA.  
-    Cấu hình và quản lý Azure AD Connect, Azure AD DS.  
-    Cài đặt mới AD DS forest trên Azure VNet.  
•    Giải Pháp Sao Lưu và Khôi Phục:  
-    Triển khai hybrid backup và recovery với Windows Server IaaS.  
-    Bảo vệ hạ tầng với Azure Site Recovery, Azure Backup.  
-    Migrate Windows Server roles, Storage Migration Service.  
Khóa học nhằm trang bị cho học viên khả năng vận hành hiệu quả các hệ thống Windows Server, Azure và các công nghệ Hybrid, từ đó chuẩn bị cho các chứng chỉ của Microsoft và phát triển sự nghiệp trong lĩnh vực quản trị hệ thống và điện toán đám mây.

## **3. Nội dung khóa học**

### **Phần I: MCSA 2022**

 1.1. Thực hành: Thời lượng thực hành chiếm 70% trong quá trình học.  
-    Thực hành trực tiếp trên máy tính có cấu hình: core-i, ram 16GB, ổ cứng SSD.  
-    Sử dụng công cụ ảo hóa Vmware Workstation Pro để thực hành. (với nền tảng ảo hóa để phù hợp với môi trường và xu thế điện toán đám mây – ảo hóa hiện tại)  
-    Trên tất cả máy ảo triển khai mô hình toàn bộ gồm các máy chủ Wimdows Server 2022 và windows 10 pro để đảm bảo những công nghệ mới nhất và phù hợp với các mô hình doanh nghiệp đang sử dụng bên ngoài.

 1.2. Nội dung  
1. Cài đặt và Cấu hình Windows Server 2022  
•    Tổng quan về Windows Server 2022  
•    Cài đặt Windows Server 2022 (Server Core, Server Full)  
•    Quản lý Windows Server bằng Windows Admin Center  
•    Cấu hình Remote Management & Windows PowerShell  
2. Quản lý File Server và Storage  
•    Cấu hình và quản lý Storage Spaces & Storage Pools  
•    Quản lý File Server Resource Manager (FSRM) & Data Deduplication  
•    Cấu hình và triển khai Distributed File System (DFS-R, DFS-N)  
•    Cấu hình quyền truy cập với NTFS & Access-Based Enumeration (ABE)  
3. Quản lý Active Directory (AD DS) & Group Policy  
•    Triển khai Active Directory Domain Services (AD DS)  
•    Quản lý Users, Groups, OU và Group Policy  
•    Cấu hình Read-Only Domain Controller (RODC)  
•    Cấu hình AD Sites & Replication  
•    Quản lý FSMO Roles & AD Trusts  
4. Dịch vụ Mạng & Bảo mật  
•    Cấu hình và quản lý DHCP (Failover, Relay Agent, Reservation)  
•    Cấu hình DNS (Zones, Conditional Forwarders, Round Robin, Netmask Reordering)  
•    Cấu hình Network Policy Server (NPS) & VPN (Site-to-Site, Remote Access)  
•    Quản lý tường lửa Windows Defender & Security Baseline  
5. Ảo hóa & High Availability  
•    Cấu hình và triển khai Hyper-V (VMs, Snapshots, Live Migration)  
•    Cấu hình Failover Clustering & Network Load Balancing (NLB)  
•    Cấu hình Storage Replica & Cluster-Aware Updating  
•    Triển khai Windows Server Containers & Docker  
6. Chứng thực & Bảo mật hệ thống  
•    Cấu hình Active Directory Certificate Services (AD CS)  
•    Cấu hình Active Directory Federation Services (AD FS)  
•    Cấu hình Active Directory Rights Management Services (AD RMS)

### **Phần II: Azure Administrator Associate**

2.1. Thực hành: Thời lượng thực hành chiếm 70% trong quá trình học.  
-    Thực hành trực tiếp trên Tài khoản đăng kí Azure của Microsoft.  
-    Sử dụng môi trường nền tảng web Azure Portal của Microsoft để thực hành.  
2.2. Nội dung:  
1. Quản lý Azure Infrastructure  
•    Đăng ký và quản lý tài nguyên trên Azure  
•    Quản lý Azure Resource Groups & Role-Based Access Control (RBAC)  
•    Giám sát hiệu suất & chi phí sử dụng Azure  
2. Quản lý Azure Storage  
•    Cấu hình Azure Storage Accounts & Blob Storage  
•    Quản lý Azure File Sync & Storage Replication  
•    Thực thi Azure Backup & Azure Site Recovery (ASR)  
3. Quản lý Virtual Machines (VMs)  
•    Triển khai và quản lý Windows/Linux VMs trên Azure  
•    Cấu hình Automated VM Scaling & VM Backup  
•    Tích hợp Hybrid VMs với Windows Admin Center & Azure Arc  
4. Quản lý Mạng Azure  
•    Cấu hình và quản lý Azure Virtual Networks (VNet)  
•    Kết nối On-Premises với Azure bằng VPN Gateway  
•    Cấu hình Azure Load Balancer & Network Security Groups (NSG)  
•    Quản lý Azure Private DNS & Public DNS  
5. Hybrid Identity & Security  
•    Triển khai Azure Active Directory (Azure AD)  
•    Cấu hình Hybrid AD với Azure AD Connect & Seamless SSO  
•    Cấu hình Azure Multi-Factor Authentication (MFA)  
•    Giám sát và bảo vệ Azure với Microsoft Defender for Cloud
